# EKA 3D Game

A browser-based 3D maze game built with vanilla JavaScript and CSS 3D transforms.

## Overview
This project creates a simple first-person 3D environment without using heavy game engines.  
The world is generated dynamically from JavaScript objects and rendered using HTML elements with CSS3 transforms.


## Features
- Procedural world creation
- CSS3 3D rendering
- Mouse pointer lock camera
- Real-time movement system
- Gravity and jumping physics
- Dynamic textures generated with CSS gradients
- HUD system with live player coordinates
